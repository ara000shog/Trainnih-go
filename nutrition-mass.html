План для новичков

Первый день(всегда 4ри приема пищи)

Овсянка на молоке + 3 яйца + арахисовая паста

Рис + куриная грудка + брокколи + оливковое масло

Протеиновый шейк + банан + горсть орехов

Гречка + лосось + овощи на пару

Второй день(всегда 4ри приема пищи)

Омлет из 4 яиц + хлеб цельнозерновой + авокадо

Картофель запечённый + говядина тушёная + овощи

Творог + мёд + горсть грецких орехов

Паста из твёрдых сортов + курица + салат

Третий день(всегда 4ри приема пищи)

Овсянка на воде с ягодами + 2 яйца всмятку + орехи

Рис басмати + индейка + овощи гриль

Протеиновый батончик + яблоко + миндаль

Гречка + лосось или форель + салат из огурцов и помидоров

Четвертый день(всегда 4ри приема пищи)

Творог с фруктами + горсть орехов + 1 варёное яйцо

Булгур + курица + овощи + оливковое масло

Протеиновый шейк + банан + арахисовая паста

Картофель пюре + говядина + салат

Пятый день день(всегда 4ри приема пищи)

Омлет из 3 яиц + овсянка с молоком + немного мёда

Рис + лосось + овощи на пару

Йогурт натуральный + орехи + фрукты

Гречка + куриная грудка + овощи

Шестой день день(всегда 4ри приема пищи)

Творог + банан + ложка мёда + миндаль

Макароны твёрдых сортов + индейка + салат из овощей

Протеиновый шейк + яблоко + горсть орехов

Булгур + говядина + овощи гриль

Седьмой день день(всегда 4ри приема пищи)

Овсянка с молоком + ягоды + 2 яйца

Картофель запечённый + лосось + овощи

Творог + мёд + орехи

Гречка + курица + салат из овощей
